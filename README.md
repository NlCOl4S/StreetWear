## 📖 Descripción
**StreetWear** es una API RESTful desarrollada con **Spring Boot** diseñada para gestionar el backend de una tienda de comercio electrónico (*e-commerce*) enfocada en ropa urbana. El sistema implementa una arquitectura robusta, control de persistencia con bases de datos relacionales y un sistema de seguridad avanzado para la autenticación de usuarios.

---

## 🚀 Características Principales
- **Autenticación y Autorización Segura:** Implementación de JSON Web Tokens (JWT) con Spring Security.
- **Gestión de Sesiones y Tokens:** Control avanzado de tokens en base de datos (`expired` y `revoked`) para permitir revocación de sesiones e invalidación de tokens activos.
- **Gestión de Catálogo:** Administración de productos, stock y categorías de ropa urbana.
- **Arquitectura Limpia:** Separación de responsabilidades mediante controladores, servicios, repositorios y entidades.

---

## 🛠️ Tecnologías Utilizadas
- **Java 17+**
- **Spring Boot**
- **Spring Security**
- **Spring Data JPA / Hibernate**
- **MySQL / Base de datos Relacional**
- **JWT (JSON Web Token)**
- **Maven** (Gestor de dependencias)
